# Adjective Atlas

Live version: [https://adjective-atlas.mypersonallearning.eth.limo/](https://adjective-atlas.mypersonallearning.eth.limo/)

## Purpose

This project is a structured vocabulary reference focused on adjectives.

The page helps readers and writers explore adjective families across major semantic domains (emotion, character, aesthetic, epistemic, social, sensory, and tonal), with:

- an interactive Plutchik emotion wheel for primary emotions and dyads,
- searchable adjective sets and contrasts,
- table and card-based organization for fast lookup and comparison.

The goal is to make nuanced word choice easier by showing relationships between terms (intensity, opposition, and conceptual proximity), not just isolated word lists.
